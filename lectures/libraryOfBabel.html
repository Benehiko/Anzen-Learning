THE LIBRARY OF BABEL 4 OF 12

Imagine the following:

A computer programme creates every possible combination of letters. When done, it will contain all possible combinations of all possible letters. This will be a huge collection of rubbish. But it will also contain every book that has ever been written.

This is what the Library of Babel is attempting. On its web site (https://libraryofbabel.info) the library is explained as follows:

The Library of Babel is a place for scholars to do research, for artists and writers to seek inspiration, for anyone with curiosity or a sense of humor to reflect on the weirdness of existence - in short, it’s just like any other library. If completed, it would contain every possible combination of 1,312,000 characters, including lower case letters, space, comma, and period. Thus, it would contain every book that ever has been written, and every book that ever could be - including every play, every song, every scientific paper, every legal decision, every constitution, every piece of scripture, and so on. At present it contains all possible pages of 3200 characters, about 10 [to the power of] 4677 books. [That is a 1 with 4 677 zero's]

Play around:

This is still a work in progress, but you can already search some of the texts that has been generated. You can take any piece of written work in any language below 3200 characters, and search for it in the massive online database. Try the following:

  • Open the library of babel's site in a new tab (https://libraryofbabel.info).
  • Search for the following text: "Johann Smith is the best lecturer in the world and he really likes coffee". You will find that this line appears in a book with a lot of rubbish words (with 410 pages).
  • Take any text in any language (as long as it is less than 3 200 characters, and search for it. You will find that these words do appear somewhere in the collection.

How does it work?

John thinks of a three-letter word. Mary uses a computer programme to create every possible three letter-word that can exist in English. The programme creates the following words: aaa, aab, aac, aad, aae, aaf, aag, aah, aai, aaj, aak, aal, aam, aan, aao, aap, aaq, aar, aas, aat, aau, aav, aaw, aax, aay, aaz, aba, abb, abc, abd, abe, abf, abg, abh, abi, abj, abk, abl, abm, abn, abo, abp, abq, abr and so forth. This list of the words will end with zzz. Now, although Mary doesn't have any idea what John's three letter word is, it will appear somewhere in this list of words.

Now, if John thinks about 3200 characters, those characters are bound to be in the current library of babel.

How does it relate to this lecture?

The amazing part about the library of babel is that it will contain the every book that has ever been written. The downside is that it will contain billions op pages that only consist of rubbish text. Finding the text your looking for may be impossible, without the search function. This is an example of valuable information hidden within an overload of rubbish. It is often easy to overwhelm users of your web site with too much information. Try to avoid this through the mechanisms explained later in this lecture.
