Where are we now?

We have now looked at the first two elements of information architecture. We will be looking at the third, navigation.

What is navigation?

In simple terms, navigation refers to the way someone can work or move through information. On a web site, this refers to the way a user moves to and form different web pages without "getting lost".

The four questions

When designing a web site, always keep the user in mind. The user will typically ask the following questions on any page:

  • Where am I at present?
  • Where can I go to from here?
  • Where have I come from?
  • What places are there to go to?

You, as the designer, will have to answer these questions.

How do we do it?

The following methods can help a lot with making navigation easier:

Site maps: This is a single page on a web site that contains the names of (and preferably the links to) all other pages on the site. It provides the user with an overview of the whole site.
