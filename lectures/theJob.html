What would you do?

The class notes (on Moodle) provides a summary of what you would be doing if you would employed as an information archictect. You would typically:

  • be responsible for creating a structure for the information that needs to be displayed on the web site.
  • to organise information in such a way that users can find the most frequently used parts easily and quickly.
  • make it possible for users to find new content quickly.
  • find the balance: Make the site easy to use for inexperienced users, but not frustrating to experienced onces.
  • make it possible to search the site.
